Stream slope (continuous)

This tool can be used to calculate the slope of stream grid cells in a raster stream network. A slope value is calculated for each stream cell, rather than the stream link slope. The background (i.e. non-stream) value of the output raster can be set to zero or NoData optionally.

See Also:

  • Stream Link Slope

Scripting:

The following is an example of a Python script that uses this tool:

wd = pluginHost.getWorkingDirectory()
streamsFile = wd + "streams.dep"
demFile = wd + "DEM.dep"
pointerFile = wd + "D8 pointer.dep"
outputFile = wd + "output.dep"
nodataBackground = "true"
args = [streamsFile, demFile, pointerFile, outputFile, nodataBackground]
pluginHost.runPlugin("StreamSlopeContinuous", args, False)
